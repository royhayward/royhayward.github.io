Open Integration Architecture is a framework. The ultimate goal is to discover the way to make Event-Driven Architecture (EDA) better. I believe that we have reached this turning point where we need to take the next step in the evolution of system architectures.

An effort to prove this out is going on in the Physical Security industry and you can find out more about it here, "Next Phase of Orchestrated Security Solutions"
